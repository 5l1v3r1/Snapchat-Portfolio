Flashcards


This Flashcards program was written in Python nearly three years ago. This program used tkinter to display the GUI to a mock-flashcard program. You can create a new Flash card by typing into the text-entries, which then is saved to a file. The program can then read from the file and display the customized flash cards using tkinter.
